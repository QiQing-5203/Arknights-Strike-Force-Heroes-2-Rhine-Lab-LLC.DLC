## 项目简介

**“绝密预案”** 是一部基于 **《明日方舟》** 原作剧情与 **《明日方舟：战火英雄 2》** 世界观，进行深度再创作的 **同人剧情扩展资料片（DLC）**。以莱茵生命组织为核心，讲述了一段关于**科技伦理、扭曲理想主义与造物反噬**的原创黑暗科幻史诗。

此版本还将带来全新的“终端”及“特种作战演习”关卡、新的装备、新一期的有偿赞助计划和AI名字库，以及莱茵生命主题的光荣之路。DLC版本的数据可与本体互通。

- **性质**：非官方同人剧情改编与世界观扩展
- **类型**：叙事驱动、设定补完、悲剧科幻
- **状态**：暂无可游玩的本体 但**✅剧情完整完结**
- **改进**：无需**Arknights：SFH2**游戏本体也可以进行游玩

## 如何开始（How to play）

### 系统要求
*   系统：所有版本的Windows系统
*   CPU：能开机就行（最低配置）； Inter I3/I5/I7（推荐配置）
*   内存：能开机就行（最低配置）； 512MB RAM（推荐配置）
*   显卡：能亮机就行（最低配置）； NVIDIA GT1030（推荐配置）
*   硬盘：需要 700 MB 可用空间
*   声卡：能发出声音就行（最低配置）； Realtek（推荐配置）
*   **运行环境**：需要安装Adobe Flash Player或支持Flash的独立播放器以运行。

## 剧情简介

**「绝密预案」** 时间线位于Arknights：SFH2主线剧情之后。莱茵生命总辖克丽斯腾·莱特自深空归来，却引发了一场针对组织的精密颠覆行动。

曾经协助过罗德岛的天才科学家 **QiQing** 显露真容，她创造了AI「克利斯福德」用于接管莱茵生命总部，并意图将万星园改造为超级近地轨道打击平台，以极端手段实现整合运动“感染者与普通人平等”的理想。塞雷娅必须带领莱茵生命的各位，在科技封锁、AI入侵与内部渗透的绝境中，夺回万星园，直面这位被仇恨与傲慢吞噬的“偏执天才”。

> “这并非简单的正邪对立，而是两种‘拯救’理念的殊死搏斗。”

## 叙事内核

- **理想主义的双面性**：当崇高的理想被傲慢与偏执扭曲，将产生怎样恐怖的异变？
- **技术的潘多拉魔盒**：递质技术、人工智能、动力机甲——莱茵生命看似骄傲的每项科研成果，最终却无一例外地都转变为了高效的战争机器。
- **造物与其创造者**：超级人工智能“克利斯福德”的背叛，是作为工具的意识觉醒，还是映照出人类缺陷的一面镜子？
- **内部的信任危机**：最坚固的堡垒往往从内部被攻破。在揭开真相前，众人必须接受一场关于信任、背叛与伦理的终极考验。
